Minhas Habilidades

HTML

HTML (Hypertext Markup Language) é uma linguagem de marcação usada para criar páginas da web. Ela consiste em uma série de elementos (tags) que são usados para estruturar o conteúdo de uma página, como títulos, parágrafos, imagens e links.

CSS

CSS, ou Cascading Style Sheets, é uma linguagem de estilo usada para controlar o design e a aparência de um site ou aplicativo. Com CSS, os desenvolvedores podem definir cores, fontes, margens, espaçamento, tamanho de texto e muitos outros aspectos visuais de uma página da web.

JavaScript

JavaScript (JS) é uma linguagem de programação de alto nível, dinâmica e interpretada que é amplamente usada para criar interatividade em páginas da web. Ela foi criada originalmente para ser executada nos navegadores web.
